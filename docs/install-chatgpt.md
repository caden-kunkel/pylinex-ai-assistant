# Using pylinex with ChatGPT

```bash
conda env create -f environment.yml
conda activate pylinex_env
```

**Option A — Custom GPT:** [pylinex-ai-assistant](URL_COMING_SOON) — click, open a chat, start asking.

**Option B — Paste prompt:** Copy [`starter_prompt.md`](../starter_prompt.md) and paste it as your first message in any ChatGPT chat.
